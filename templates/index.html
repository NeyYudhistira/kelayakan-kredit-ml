Keunggulan Sistem

Dirancang untuk keputusan kredit yang lebih baik

Setiap komponen dioptimalkan untuk akurasi, transparansi, dan kemudahan penggunaan oleh analis kredit.

🎯

Prediksi Presisi Tinggi

Random Forest dengan 300 pohon keputusan dan class_weight="balanced" memastikan prediksi yang adil untuk semua profil nasabah.

⚖️

Penanganan Ketidakseimbangan Data

Teknik SMOTE menyeimbangkan distribusi kelas sebelum pelatihan, sehingga model tidak bias terhadap kelas mayoritas.

📊

18 Fitur Analisis

Mencakup skor kredit, rasio hutang, riwayat pembayaran, aset tabungan, profil pekerjaan, tujuan pinjaman, dan lebih banyak lagi.

🔒

Normalisasi Data Otomatis

StandardScaler memastikan semua fitur numerik berada pada skala yang sama, mencegah dominasi fitur berskala besar.

📈

Tingkat Keyakinan Model

Setiap prediksi disertai confidence score berdasarkan probabilitas voting dari seluruh pohon keputusan dalam ensemble.

⚡

Respons Real-time

Model terlatih dimuat langsung dari file .pkl sehingga prediksi dihasilkan dalam hitungan milidetik tanpa pelatihan ulang.

Stack Teknologi

Dibangun di atas fondasi yang terpercaya

Kombinasi library machine learning industri standar untuk performa dan reproduktabilitas optimal.

🌲

Random Forest

Ensemble 300 Decision Trees · max_depth=15 · min_samples_split=5 · class_weight balanced

🔄

SMOTE

Synthetic Minority Oversampling diterapkan hanya pada data training, bukan test set.

📐

StandardScaler

Scaling setelah SMOTE untuk menghindari kebocoran data. Fit pada train, transform pada test.

🐍

Flask & Scikit-learn

Backend Flask menyajikan model .pkl + scaler + label_encoders secara real-time.
