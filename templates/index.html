Tentang KreditSmart

Platform analisis kelayakan kredit berbasis AI yang membantu lembaga keuangan membuat keputusan pemberian pinjaman yang lebih akurat dan adil.

Prediksi Akurat

Menggunakan algoritma Random Forest dengan akurasi tinggi untuk menilai risiko kredit berdasarkan profil keuangan nasabah secara komprehensif.

Analisis Menyeluruh

Mempertimbangkan 18 fitur meliputi skor kredit, riwayat pembayaran, rasio hutang, aset tabungan, dan profil pekerjaan nasabah.

Keputusan Transparan

Setiap prediksi disertai tingkat keyakinan model dan rekomendasi tindak lanjut yang jelas bagi analis kredit maupun nasabah.

Teknologi yang Digunakan

Stack teknologi modern dan terpercaya untuk performa optimal

Random Forest

Algoritma ensemble berbasis Decision Trees dengan teknik Bagging. Membangun ratusan pohon keputusan secara paralel lalu mengambil voting mayoritas untuk hasil prediksi yang robust.

SMOTE

Teknik oversampling untuk menangani ketidakseimbangan kelas pada data kredit, sehingga model tidak bias terhadap kelas mayoritas.

Scikit-learn

Library machine learning untuk preprocessing, training, dan evaluasi model dengan StandardScaler dan LabelEncoder.

Flask & Python

Backend ringan berbasis Flask untuk menyajikan prediksi model secara real-time melalui antarmuka web yang responsif.
